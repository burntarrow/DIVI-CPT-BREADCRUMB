A native Divi 5 breadcrumb module for hierarchical service-category terms and services permalinks.

== Description ==

Renders trails such as Home > Services > Residential > New Construction > Post Title by reading the WordPress post type and taxonomy hierarchy instead of parsing URL segments.

== Installation ==

1. Upload and activate the plugin.
2. Add the Service Breadcrumbs module to a Divi 5 layout or Theme Builder template.
3. Keep the default post type `services` and taxonomy `service-category`, unless your registered slugs differ.
