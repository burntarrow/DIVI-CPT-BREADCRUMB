# Changelog

## 0.2.0 — 2026-07-22

- Rebranded the plugin as Divi 5 CPT Breadcrumbs.
- Generalized breadcrumb resolution for any public post type and taxonomy.
- Replaced post type and taxonomy slug inputs with dynamically populated Divi select fields.
- Included public and publicly queryable post types and taxonomies, including headless registrations without an admin UI.
- Added a protected REST endpoint for registered object discovery.
- Filtered taxonomy options by their registered post types in the Visual Builder.
- Added Automatic and None data-source choices.
- Added hierarchical page and CPT ancestor trails.
- Preserved Yoast SEO and Rank Math primary-term support.
- Added PHP, metadata, markup, data-source, and Visual Builder bundle smoke tests.
